Party Script

Introduction

Party Script is a Javascript bookmarklet that turns any web page into a party.

It can be used to prank friends who forget to lock their computer. Upon activation of the bookmarklet, you will have 5 seconds before it is armed. When your victim returns to their computer and moves their mouse, the party will begin.

Usage

  1. On your victim's computer, open a new tab and navigate to this web page.
  2. Install the bookmarklet to their web browser using the button below.
  3. Go back to the tab on which you want to host the party, and click the bookmarklet to activate the script.

Settings Help

Delay

This sets the speed of the party. The smaller the number, the faster the party.

Min/Max

This sets the colors of the party.

Bookmark Name

This sets the name of the bookmark when it is added to the browser's bookmarks.

Party Mode

Instant Party mode kicks off the party instantly.

Surprise Party mode kicks off the party when the guest of honor arrives (when the mouse cursor is moved). This mode is ideal for setting up on a friend's computer while he's away from his desk. The script gives you a 5-second getaway window before it starts watching for mouse movements.
